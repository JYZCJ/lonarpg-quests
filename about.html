About This Website

This website was made for an assignment in MI 103. I chose LonaRPG Repeatable Quest Rating as the topic because when I looked through the LonaRPG wiki, I found that it mostly only explains the quests themselves. Because of that, it can be a little hard to tell which quests are the best ones to do. That is why I decided to make this website about that topic. I really like this game, but when I first started playing, I was pretty confused. I hope this website can help other players :)
